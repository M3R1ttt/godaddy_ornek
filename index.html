Web hosting nedir ve nasıl çalışır?

Web hosting, internette erişilebilir olmaları amacıyla web sitenizin dosyalarını (fotoğraflar, videolar, metin, kod vb.) kaydetmek için ihtiyacınız olan depolama alanıdır. Bazıları web hosting’i, bir evin üzerine inşa edildiği arsaya benzetir. Paylaşılan hosting ile, hizmet sağlayıcınız (yani web host) size bir sunucu üzerinde, web dosyalarınızın (ev/web sitesi) depolanabileceği bir alan (arsa) satar veya kiralar. Web host'ları hosting hizmetlerinizle ilişkilendirilebilecek bir alan adına sahip olmanızı bekler ve gerekirse bir alan adı satın almanıza yardımcı olabilirler.


Web hosting planı satın aldıktan sonra, sitenizi sunucularımızda depoluyor ve ona benzersiz bir DNS atıyoruz. DNS, dünyanın her yerindeki insanların web sitenize erişmesini sağlayan adrestir. Bu benzersiz adres, insanların sitenizi görüntülemesi için gereklidir.


Bu sayfada gösterildiği gibi web site hosting (paylaşımlı hosting) paketi satın alarak aslında sunucularımızdan alan satın almış olursunuz. Bu, bir bilgisayarın sabit sürücüsündeki alana benzer. Ayrıca sunucumuzda olmanız, web sitenizin dosyalarına her yerden erişilebilmesini sağlar.

cPanel Hosting nedir ve ne gibi avantajları vardır?

cPanel, web hosting müşterisine/sahibine web sitelerini ve paylaşımlı sunucularını kolayca yönetmelerine yardımcı olacak bir kullanıcı arayüzünü sunan bir web hosting kontrol panelidir. WordPress, Drupal, Magento ve Joomla gibi çeşitli içerik yönetimi sistemlerini (CMS) kullanarak kullanıcıların web siteleri oluşturmalarına olanak tanır.


cPanel hosting ile, sektör standardı kontrol paneliyle aşağıdakiler dahil olmak üzere ihtiyacınız olan tüm barındırma özelliklerine ve ayarlara erişin:




CloudLinux

Dengeli CPU, RAM ve Disk G/Ç sınırlarıyla sitenizin çevrimiçi kalmasını sağlayın.

İsteğe Bağlı Kaynaklar

Tek tıklamayla daha fazla CPU, RAM, Giriş İşlemi ve G/Ç edinin.

Cage FS

Sanallaştırılmış dosya sistemiyle web sitenizin içeriğini koruma altına alın.

Ücretsiz uygulamalar

Yalnızca tek tıklamayla web sitenize 125'ten fazla uygulamadan dilediğinizi yükleyin.

Mevcut web sitemi, GoDaddy paylaşımlı web hosting hizmetine taşıyabilir miyim?

Evet ve bunu ücretsiz yapabilirsiniz. Mevcut web sitenizi GoDaddy Web Hosting hizmetine taşımak oldukça kolay bir işlemdir. Yeni hesap katılım işleminin bir parçası olarak, ücretsiz Otomatik Site Taşıma Aracımızı kullanarak mevcut sitenizi, GoDaddy Web Hosting hesabınıza geçirmeyi seçebilirsiniz. Birkaç basit soruyu yanıtlayın ve sitenizi taşıyalım. Bu hosting hesabınızı taşıma rehberinde verilen adım adım talimatları da izleyebilirsiniz.


Herhangi bir sorunuz olması halinde, uzman destek ekibimiz size yardım için yanınızdadır. Web sitenizi GoDaddy hesabına dönüştürmeye yönelik en iyi yöntemi belirlemenize yardımcı olabiliriz.

Web sitem büyürse ve daha fazla alana ihtiyacım olursa web hosting planımı değiştirebilir miyim?

Evet. Daha fazla işleme gücüne veya depolama alanına ihtiyacınız olduğunda kolayca yeni bir web hosting planına yükseltebilirsiniz. Planınızı yükseltmek trafiğiniz artarken web sitenizi daha hızlı ve daha esnek hale getirebilir.

Paylaşılan hosting planıma dahil olan alan adını ve SSL’yi nasıl edineceğim?

Tercih ettiğiniz Web Hosting planını seçtiğinizde, ücretsiz SSL sertifikanız sepetinize otomatik olarak eklenir. Sepetinize devam ettiğinizde satın alma işleminize ekleyecek bir alan adı arama seçeneğiniz olur. Ayrıca alan adları sayfamızda kullanılabilir domain sorgulama yapabilir ve burada sepete ekleyebilirsiniz.


ÖNEMLİ: Web Hosting planınızın satın alma işlemini tamamlamadan önce alan adını sepetinize eklediğinizden emin olun.

AutoSSL nedir ve nasıl çalışır?

AutoSSL ücretsizdir ve ücretli SSL sertifikalarına alternatif olarak hosting planınıza dahil edilir. GoDaddy Web Hosting planı satın aldığınızda AutoSSL, cPanel hosting hesabınızda oluşturulan veya yönetilen web sitelerine Sectigo tarafından bir DV SSL Sertifikasını otomatik olarak atar. SSL sertifikanız güncel kalır ve sizin adınıza otomatik olarak yenilenir. Böylece, GoDaddy Web Hosting planınızı aktif tuttuğunuz sürece sertifikanız konusunda endişelenmenize gerek kalmaz.

Kredi kartlarını işleyen web sitelerinin ve uygulamaların PCI uyumlu olması beklenmektedir.

Ödeme Kartı Sektör Güvenliği Standartları Konseyi (PCI-SSC), kredi kartı verilerini koruyan standartları belirler. Yani kredi kartı bilgilerini ileten, işleyen veya depolayan işletmelerin Ödeme Kartı Sektör Veri Güvenliği Standartları’nda belirtilen kurallara uyması beklenmektedir. Web siteniz aracılığıyla kredi kartı ödemelerini kabul etmek için müşterilerinizin ödeme bilgilerinin güvende olduğundan ve PCI gereksinimlerini karşıladığınızdan emin olmanız gerekir. Bu süreci kolaylaştırmak amacıyla PayPal Checkout, Square Online Checkout ve Stripe Checkout gibi üçüncü taraf ürünleri web sitenize ekleyerek ödemeleri adınıza işleme almalarını (ve hosting sunucunuzda kredi kartı verilerinin bulunmamasını) sağlayabilirsiniz. İşinizin PCI uyumlu kalmasını sağlamak için ek gereksinimleri bildiğinizden emin olun.

Ödemeleri doğrudan sunucunuz üzerinden işleme almak isterseniz WooCommerce Hosting gibi PCI sertifikalı ürünler sunuyoruz.
